rosy_

Paste in a GitHub PR, get back the commit history it deserved.

Go Claude AI MIT

rosy looks at a GitHub PR and asks Claude to redraft its commit history into what should have been there from the start.

Step through the redrafted commits in a pretty TUI. If you actually want to rewrite history, press i — but that's probably crazy.

#Install

Homebrew
$ brew install zhubert/tap/rosy
Go
$ go install github.com/zhubert/rosy@latest

#Usage

~/Code/rosy — zsh
$ rosy https://github.com/owner/repo/pull/123 rosy: opening the PR ...0.8s rosy: sizing up the diff (4 files, +50 / -462) ...1.2s rosy: ghost-writing better commits via sonnet ...48.3s rosy: LGTM
